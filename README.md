## 💻 استخدام DatabaseManager

### مثال 1: تسجيل الدخول
```python
from database_manager import DatabaseManager

db = DatabaseManager()

# محاولة تسجيل دخول
user = db.authenticate_user("admin@posapp.com", "123456")

if user:
    print(f"مرحباً {user['name']} - الدور: {user['role_name']}")
else:
    print("بيانات غير صحيحة")

db.close()
```

### مثال 2: إضافة منتج
```python
db = DatabaseManager()

# إضافة منتج
product_id = db.add_product(
    product_code="P001",
    product_name="هاتف ذكي",
    category_id=1,  # إلكترونيات
    buy_price=800.0,
    sell_price=1000.0,
    barcode="123456789"
)

db.close()
```

### مثال 3: إنشاء فاتورة
```python
db = DatabaseManager()

# إنشاء فاتورة
invoice_number = db.create_invoice(
    store_id=1,
    cashier_id=2,
    customer_name="محمد علي",
    customer_phone="01012345678"
)

# إضافة منتج للفاتورة
db.add_invoice_item(
    invoice_number=invoice_number,
    product_id=1,
    quantity=2,
    unit_price=1000.0,
    discount=50.0
)

db.close()
```

## 🌟 أهم الميزات الجديدة

### 1. تقرير مبيعات الأصناف اليومي (للمديرين)
- عرض تفصيلي للأصناف المباعة يومياً لكل فرع.
- فلاتر متقدمة (تاريخ، فرع، صنف).
- إمكانية استخراج البيانات بصيغة **Excel** و **PDF**.
- متاح فقط لمدير النظام والمطور لضمان سرية البيانات.

### 2. اختصارات لوحة المفاتيح
| الاختصار | المهمة |
| :--- | :--- |
| `F1` - `F12` | التنقل السريع بين التبويبات |
| `F5` | تحديث البيانات الحالية |
| `Ctrl + R` | الانتقال السريع لمرتجع الفواتير |
| `Ctrl + F` | التركيز على حقل البحث في الكاشير |
| `F12` | إتمام عملية البيع (الدفع) |
| `Ctrl + S` | تعليق الفاتورة |
| `Delete` | حذف صنف من الفاتورة |
| `Ctrl + L` | تسجيل الخروج |

### 3. تحسينات الأمان
- إخفاء **سعر الشراء** (Buy Price) تماماً عن الموظفين ومديري الفروع، وقصره على مدير النظام والمطور فقط.
