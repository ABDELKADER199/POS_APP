# نظام نقطة البيع المتكاملة (POS)
# Integrated Point of Sale System

## 📋 نظرة عامة

نظام متكامل لإدارة المبيعات والمخزون مع واجهة رسومية حديثة وسهلة الاستخدام، مبني بـ PyQt6 وقاعدة بيانات MySQL.

## ✨ المميزات الرئيسية

### 1. **تسجيل الدخول الآمن** 🔐
- تشفير كلمات المرور بـ bcrypt
- حماية من هجمات Brute Force (5 محاولات فقط)
- واجهة سهلة الاستخدام

### 2. **إدارة المستخدمين** 👥
- إضافة وتعديل وحذف المستخدمين
- 5 أدوار مختلفة:
  - **Admin**: الوصول الكامل
  - **Manager**: معظم الصفحات
  - **Cashier**: البيع والدرج
  - **Call Center**: إدارة الأوامر
  - **Warehouse Staff**: المخزون

### 3. **نقطة البيع (POS)** 🛍️
- بحث سريع عن المنتجات
- إضافة منتجات للفاتورة
- حساب تلقائي للإجماليات
- طرق دفع متعددة (نقد، بطاقة، شيك)
- طباعة الفواتير

### 4. **إدارة المنتجات والمخزون** 📦
- إضافة منتجات جديدة
- تعديل الأسعار والمعلومات
- استيراد من ملف Excel
- إدارة المخزون حسب الفرع
- تنبيهات المخزون المنخفض

### 5. **إدارة أوامر العملاء** ☎️
- تسجيل بيانات العميل
- إنشاء أوامر جديدة
- توجيه الأوامر للفروع المختلفة
- تتبع حالة الأوامر

### 6. **إدارة الدرج النقدي** 💰
- فتح الدرج مع رصيد افتتاحي
- إدخال الفئات النقدية (200، 100، 50، 20، 10، 5، 1، 0.5 ج.م)
- إغلاق الدرج مع التحقق من الرصيد
- تنبيهات الفروقات

## 📥 التثبيت والإعداد

### 1. تثبيت المكتبات:
```bash
pip install -r requirements.txt
```

### 2. إعداد قاعدة البيانات:
```bash
# تأكد من تشغيل MySQL
mysql -u root -p

# ثم قم بتشغيل سكريبت الإعداد
python setup_simple.py
```

### 3. تشغيل التطبيق:
```bash
python main.py
```

## 🚀 الاستخدام السريع

### تسجيل الدخول:
1. افتح التطبيق
2. أدخل البريد الإلكتروني وكلمة المرور
3. انقر "تسجيل الدخول"

### إنشاء فاتورة:
1. انتقل إلى صفحة الكاشير
2. ابحث عن المنتجات
3. أضفها إلى الفاتورة
4. انقر "إتمام البيع"

### إضافة منتج:
1. افتح صفحة المنتجات
2. انقر "إضافة منتج جديد"
3. أدخل البيانات
4. انقر "حفظ"

### فتح الدرج النقدي:
1. افتح صفحة الدرج
2. أدخل الرصيد الافتتاحي
3. انقر "فتح الدرج"
