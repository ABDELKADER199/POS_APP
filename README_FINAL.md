# 🎉 نظام نقطة البيع المتكاملة - POS System

## ✅ المشروع مكتمل بنجاح!

نظام متكامل لإدارة المبيعات والمخزون مع واجهة رسومية حديثة وآمنة.

---

## 🚀 البدء السريع

### 1️⃣ تثبيت المكتبات
```bash
pip install -r requirements.txt
```

### 2️⃣ إعداد قاعدة البيانات
```bash
python setup_simple.py
```

### 3️⃣ تشغيل التطبيق
```bash
python main.py
```

## ✨ المميزات الرئيسية

✅ **التسجيل الآمن**: تشفير bcrypt وحماية من Brute Force
✅ **إدارة المستخدمين**: 5 أدوار مختلفة مع صلاحيات محددة
✅ **نقطة البيع**: فواتير فورية مع حساب تلقائي
✅ **إدارة المنتجات**: استيراد من Excel وتعديل الأسعار
✅ **إدارة المخزون**: تتبع الكميات لكل فرع
✅ **إدارة الأوامس**: تسجيل وتتبع أوامس العملاء
✅ **إدارة الدرج**: فتح وإغلاق مع فئات نقدية

---

## 📊 الملفات المرفقة

- **main.py** - نقطة البداية
- **database_manager.py** - إدارة قاعدة البيانات
- **ui/** - واجهات المستخدم (7 صفحات)
- **SYSTEM_README.md** - دليل شامل
- **DATABASE_DOCUMENTATION.md** - توثيق قاعدة البيانات

---

## 🔧 المتطلبات

- Python 3.10+
- MySQL 8.0+
- PyQt6, mysql-connector-python, bcrypt, openpyxl
