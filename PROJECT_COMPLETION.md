## 🔄 سير العمل

### تسجيل الدخول:
1. المستخدم يدخل بيانات تسجيل الدخول
2. التطبيق يتحقق من البيانات في قاعدة البيانات
3. التطبيق يفك تشفير كلمة المرور باستخدام bcrypt
4. إذا كانت صحيحة، يتم عرض لوحة المعلومات
5. إذا كانت خاطئة، يتم عرض رسالة خطأ وزيادة عداد المحاولات

### إنشاء فاتورة:
1. الكاشير يبحث عن منتج
2. يضيف المنتج للفاتورة
3. النظام يحسب الإجمالي تلقائياً
4. الكاشير يختار طريقة الدفع
5. يتم إنشاء الفاتورة وتحديث المخزون

### فتح/إغلاق الدرج:
1. الكاشير يفتح صفحة الدرج
2. يدخل الفئات النقدية الافتتاحية
3. النظام يحسب الإجمالي
4. عند الإغلاق، يدخل الفئات الختامية
5. النظام يقارن ويبلغ عن أي فرق
